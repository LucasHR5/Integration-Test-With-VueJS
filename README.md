# Projeto de Teste de Integração com Backend

Este repositório contém um projeto destinado a testes de integração com um backend, com o objetivo de validar a comunicação entre a aplicação cliente e a API.

O projeto é utilizado para verificar o funcionamento de requisições HTTP, o envio e recebimento de dados e o comportamento das respostas do backend durante o processo de integração.

Todo o código-fonte do projeto está disponível na branch master.
